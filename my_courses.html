HTML & JavaScript

  • Course

  • Lectured by ›
    Wang beilei
  • Tags ›
    SWC

HTML or HyperText Markup Language is the standard markup language used to create web pages.A web browser can read HTML files and compose them into visible or audible web pages. The browser does not display the HTML tags, but uses them to interpret the content of the page. HTML describes the structure of a website semantically along with cues for presentation, making it a markup language rather than a programming language.

JavaScript is a dynamic computer programming language. It is most commonly used as part of web browsers, whose implementations allow client-side scripts to interact with the user, control the browser, communicate asynchronously, and alter the document content that is displayed. It is also used in server-side network programming with frameworks such as Node.js, game development and the creation of desktop and mobile applications.

We learned about HTML, JavaScript as well as CSS in this course. We also learned how to make webpages with Dreamweaver. Dreamweaver is such a powerful tool that we can design a website in a WYSIWYG way and it indeed makes this work much easier one. In order to implement background interaction we learned a little bit about Servlet. The HTML & JavaScript course is a very interesting course. Since we only have a class every week,the course is too short for us to know all about website making.

A computer network or data network is a telecommunications network that allows computers to exchange data. In computer networks, networked computing devices pass data to each other along data connections. Data is transferred in the form of packets. The connections (network links) between nodes are established using either cable media or wireless media. The best-known computer network is the Internet.

In computer network course we learned about the five layers: The Physical Layer, The Data Link Layer, The Medium Access Control Sublayer, The Network Layer, The Transport Layer,The Application Layer as well as Network Security.

Object-oriented programming (OOP) is a programming paradigm that represents the concept of "objects" that have data fields (attributes that describe the object) and associated procedures known as methods. Objects, which are usually instances of classes, are used to interact with one another to design applications and computer programs. C++, Objective-C, Smalltalk, Delphi, Java, Javascript, C#, Perl, Python, Ruby and PHP are examples of object-oriented programming languages

We learned mainly learned Java in this course. We have already learned the prerequisite course SSD1 last semester. In this SSD3 course we learned about Objects and Classes, Object Interactions, Relationships Between Objects, Designing Classes, Advanced Class Design, File I/O, Graphical User Interface. We also have lab class for this course. I used to think that I have known all about Java and OOP after the SSD1 course, but the SSD3 course taught me not only coding but also design.
